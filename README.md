# 🎨 Pixel Art Maker

A colorful, interactive Pixel Art Studio built with vanilla HTML, CSS, and JavaScript. Users can pick colors, drag to draw on a responsive canvas, adjust grid dimensions dynamically, and clear the canvas with a single click.

## ✨ Features

- 🎨 **Custom Color Picker:** Pick any color to paint pixels.
- 🖌️ **Click & Drag Drawing:** Hold down the mouse button to draw smoothly across grid cells.
- 📐 **Dynamic Grid Resizing:** Adjust grid size on the fly from 8x8 up to 32x32.
- 🧼 **One-Click Clear:** Instantly reset the canvas to white.
- 🌈 **Vibrant UI:** Modern CSS styling with glowing glassmorphism controls and gradients.

## 🛠️ Tech Stack

- **HTML5** - Structure & Controls
- **CSS3** - Layout, Animations, & Custom Styling
- **JavaScript (ES6)** - DOM Manipulation & Event Handling
